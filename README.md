# 🤖 MyCV AI - Resume Analyzer

## 📌 Overview
MyCV AI is a smart resume analysis tool that helps users evaluate their resumes based on job-specific skills and generate an ATS (Applicant Tracking System) score. It provides insights into missing skills and suggests improvements to increase job selection chances.

---

## 🚀 Features
- 📄 Upload Resume (PDF format)
- 🎯 Job Role Selection (100+ roles available)
- 📊 Match Score Calculation
- 📈 ATS Score Prediction
- ❗ Missing Skills Detection
- 💡 Resume Improvement Suggestions
- 📉 Performance Visualization (Charts)
- 🔐 Secure Login & Signup System

---

## 🛠 Tech Stack
- **Python**
- **CustomTkinter** (Modern GUI)
- **pdfplumber** (PDF text extraction)
- **Matplotlib** (Data visualization)
- **JSON** (User database)
- **Hashlib** (Password encryption)

---

## ⚙️ How It Works
1. User logs in or creates an account  
2. Uploads a resume (PDF)  
3. Selects a job role  
4. System extracts resume text  
5. Matches resume with job skills  
6. Generates:
   - ✅ Match Score  
   - ✅ ATS Score  
   - ✅ Missing Skills  
   - ✅ Suggestions  

---

## 📊 Scoring Logic
- **Match Score** → Based on skill keyword matching  
- **ATS Score** → Combination of:
  - Match Score (60%)
  - Resume length (40%)

---

## 🔐 Security
- Passwords are encrypted using **SHA-256 hashing**
- User data stored securely in JSON format
